T129477 神奇の函数题解

题目背景题目背景题目背景

备注：此题由数学题改编，是个签到题。
题目描述题目描述题目描述
存在一个单调函数 f(x)f(x)f(x) ，符合以下限制：

输入格式输入格式输入格式
一个正整数 aaa 和一个实数 xxx 。
输出格式输出格式输出格式
f(x)f(x)f(x)，格式为 p^q 。
样例样例样例
in

1 1

out

1^1

提示与说明:

  • 有理数包括整数（正整数和负整数）和分数（以小数形式输入）。

实数包括有理数和无理数。

输入保证无理数只有可能是 π\piπ 和 k\sqrt{k}k​ 的形式。（k≥1k\ge1k≥1)

题解:

思路:
通过证明，得到
code:

#include <bits/stdc++.h>
using namespace std;
int main()
{
    string s,a;
    cin>>a>>s;
    cout<<a<<'^'<<s<<'\n';
    return 0;
}
